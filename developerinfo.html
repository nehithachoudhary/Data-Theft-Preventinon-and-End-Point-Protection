Data Theft Prevention & Endpoint Protection

This project addresses the growing threat of data theft and unauthorized access via USB ports, a major vulnerability in computer systems. Attackers can exploit USB devices to introduce malware or steal data using live boot drives. The system secures USB ports by enabling or disabling them through user authentication and OTP verification via email. It features admin and user modules, time-bound OTPs, and limited attempts to prevent brute-force attacks. By combining physical control, authentication, and user-friendly security measures, the project ensures safe USB usage in organizations, minimizing the risk of breaches, data loss, and system compromise.
